| 屬性 | 值 |
|------|-----|
| 作者 | Music Memory Team |

| 狀態 | 草稿 |

## 背景

音樂記憶 (Music Memory) 是一款結合 YouTube 音樂解析、AI 智慧推薦與社交探索的跨平台音樂應用程式。本文件定義前端 Flutter 應用的功能需求與 UI/UX 規格。

## 專案資訊

| 項目 | 內容 |
|------|------|
| 專案名稱 | music_memory_ui |
| 框架 | Flutter 3.x |
| 平台 | iOS / Android / Web |
| 狀態管理 | Riverpod |
| 認證 | Firebase Authentication (OAuth) |

---

## 頁面結構與導航

### 底部導航列

```
🏠 首頁  |  🔗 解析  |  🤖 探索  |  📋 清單  |  ⚙️ 設定
```

### 頁面清單

| 頁面 | 說明 | 優先級 |
|------|------|--------|
| 登入頁 | OAuth 登入入口（Google / Apple / Facebook）| P0 |
| 首頁 | 最近播放、快速操作入口 | P0 |
| 解析分頁 | 貼上 YouTube 網址進行解析 | P0 |
| 探索分頁 | AI 推薦滑動卡片介面 | P0 |
| 清單分頁 | 已收藏音樂列表與分類 | P0 |
| 播放器頁 | 音樂播放介面（含歌詞）| P0 |
| 設定頁 | 帳號管理、偏好設定 | P1 |

## 功能需求

### F0：用戶認證（Firebase Authentication + OAuth）

**登入頁 UI：**
- 清楚展示所有可用登入方式的按鈕
- Google 登入按鈕（所有平台）
- Apple Sign-In 按鈕（iOS / Web）
- Facebook 登入按鈕（所有平台）
- Email/Password 登入（備用）
- 匿名試用入口（可後續綁定）

**技術實現：**
- 使用 `firebase_auth` SDK 處理認證
- 使用 `google_sign_in` 處理 Google OAuth
- 使用 `sign_in_with_apple` 處理 Apple Sign-In
- 使用 `flutter_facebook_auth` 處理 Facebook OAuth
- Dio 攔截器自動注入 Firebase ID Token 至 API 請求
- Token 自動刷新，用戶無需重複登入
- 路由守衛（Auth Guard）保護需認證頁面

**驗收標準：**
- [ ] Google 登入在 iOS / Android / Web 均可正常使用
- [ ] Apple Sign-In 在 iOS / Web 可正常使用
- [ ] Facebook 登入在所有平台可正常使用
- [ ] 首次登入自動同步用戶資料至後端
- [ ] Token 自動刷新機制正常
- [ ] 登入頁面 UI 清楚展示所有登入方式

### F1：YouTube 音樂解析

**解析分頁 UI：**
- 網址輸入框（支援貼上）
- 從系統分享接收 YouTube 網址（Deep Link）
- 解析結果卡片：封面、歌名、演出者、時長
- Cover 曲標記勾選框
- 原曲同步開關
- 儲存按鈕

**驗收標準：**
- [ ] 支援從其他 App 分享 YouTube 網址至本 App
- [ ] 支援手動貼上網址
- [ ] 解析結果可手動編輯
- [ ] 解析時間 < 3 秒
- [ ] 無效網址友善錯誤提示

### F2：AI 智慧推薦（音樂探索）

**探索分頁 UI：**
- 卡片式介面，展示封面、歌名、演出者
- 右滑：喜歡（綠色動畫）→ 加入候選
- 左滑：不喜歡（紅色淡出動畫）→ 跳過
- 愛心按鈕：特效動畫 → 直接加入收藏
- 即時預覽播放按鈕

**驗收標準：**
- [ ] 滑動動畫流暢（FPS ≥ 60）
- [ ] 支援即時預覽播放
- [ ] 愛心按鈕有特效動畫
- [ ] 卡片預載不少於 20 張

### F3：音樂清單管理

**清單分頁 UI：**
- 清單視圖 / 卡片視圖切換
- 排序按鈕：發行時間、上傳時間、聽歌次數、AI 分類、是否原曲
- 篩選器：Cover 曲篩選、AI 分類篩選
- 搜尋框（歌名 / 演出者）
- 多選模式（批量刪除 / 移動）

**驗收標準：**
- [ ] 兩種視圖可自由切換
- [ ] 各排序方式可升降序
- [ ] 搜尋即時篩選
- [ ] 批量操作流暢

### F4：播放器

**播放器頁 UI：**
- 全螢幕播放介面
- 封面圖片展示
- 播放控制（上一首 / 播放暫停 / 下一首）
- 進度條（可拖曳）
- LRC 歌詞同步顯示
- Mini 播放器（底部常駐）

**驗收標準：**
- [ ] 支援背景播放
- [ ] 支援鎖屏控制
- [ ] 歌詞同步誤差 < 0.5 秒

### F5：歌詞漂浮視窗（進階功能）

- 歌詞以懸浮視窗顯示於其他 App 之上
- 可拖曳位置
- 可調整透明度
- 支援 Android 與 iOS 平台

### F6：下載與備份（進階功能）

- 下載進度條
- 離線播放支援
- 雲端備份（Google Drive / iCloud）
